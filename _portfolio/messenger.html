Built with: Swift, XCode, Firebase API, Google ML Kit iOS
This innovative messaging application breaks down language barriers by providing instant translations in real time chat conversations. The development leveraged Firebase's database for message synchronization and user authentication, while Google's ML Kit handles on-device translation between multiple languages. The app features a clean, intuitive interface that integrates translation capabilities.
