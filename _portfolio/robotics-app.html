Built with: Android Studio, Kotlin
This Android application was developed to enhance the scouting process for robotics competitions, providing teams with a powerful tool for data collection and analysis. The app enables users to record and track other teams' capabilities, match performances, and technical specifications. Developed in Kotlin using Android Studio, the application features a robust local database for offline access to team data.
